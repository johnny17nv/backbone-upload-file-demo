Backbone upload file demo
=========================

Upload file with Backbone.JS over RESTful API powered by NodeJS.

## To run

```
npm start
```

## To quit

```
Press [Enter] to close server
```

## To user

- Open [web site](http://127.0.0.1:3000)
- Drag&drop image file into the web site
